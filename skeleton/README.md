# %name

An [Electron](https://www.electronjs.org) App using [simpl4u](https://github.com/m0rtadelo/simpl4u) library.

## Getting started

1. **Init git and submodules**
  ```sh
  git init
  git add .
  git commit -m "Initial commit"
  git clone https://github.com/m0rtadelo/simpl4u simpl4u
  git submodule update --init --recursive
  ```
2. **Install dependencies**
  ```sh
  npm i
  ```

3. **Execute demo**
  ```sh
  npm start
  ````
